# Pomodoro

Focus timer with work and break cycles. 25/5/15 minute intervals. Session tracking.

**Category:** Productivity  
**Live:** [pomodoro.freeappstore.online](https://pomodoro.freeappstore.online)

## Development

```bash
pnpm install
pnpm dev
```

## Deploy

Push to main — auto-deploys to R2 via GitHub Actions.
